# Dad Jokes Command

A Jarvis command that tells random dad jokes using the icanhazdadjoke.com API.

## Features

- Fetches random dad jokes from the free icanhazdadjoke.com API
- No API key or configuration required
- Works with natural voice commands

## Usage

Say any of these voice commands:

- "Tell me a joke"
- "I want to hear a dad joke"
- "Make me laugh"
- "Tell me something funny"
- "Give me a dad joke"
- "I need a good joke"

## Installation

Install this command through the Jarvis mobile app or by adding it to your node.

## Dependencies

- `requests` (>=2.28.0) - for making HTTP requests to the joke API

## API

This command uses the free [icanhazdadjoke.com](https://icanhazdadjoke.com/) API, which doesn't require any authentication or API keys.
